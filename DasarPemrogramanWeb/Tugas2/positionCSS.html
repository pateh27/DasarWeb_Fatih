position static

Position static digunakan untuk mengatur elemen menjadi statis secara default. elemen akan mengikuti posisi normal secara default, elemen tidak dipengeruhi oleh properti seperti top, bottom, left dan right.

elemen ini diatur menggunaakn posisi : static;




position: relative

sebuah element HTML menggunakan position relative akan terletak pada posisinormal. Mengatur properti atas, kanan, bawah, dan kiri dari elemen yang diposisikan dengan "relatih" akan membuatnya jauh dari posisi normalnya. konten lain tidak akan disesuaikan agar sesuai dengan celah yang ditinggalkan oleh elemen tersebut.:

elemen ini diatur menggunaakn posisi : relative;




position: fixed

sebuah element HTML yang disetting dengan position fixed akan memiliki sifat tetap, tanpa ada perubahan bahkan jika halaman website di scroll. Berlaku pengaturan left, bottom, top, dan right pada position fixed:

elemen ini diatur menggunaakn posisi : fixed;















position: absolute

sebuah element HTML yang disetting dengan position absolute akan diposisikan relatih dengan elemen lain yang mendahuluinya yang terdekat, bukan relatif terhadap layar secara normal.

Elemen ini diatur menggunakan posisi : relative
Elemen ini diatur menggunakan posisi : absolute






position: sticky:

Cobalah untuk scroll untuk mengetahui bagaimana posisi sticky bekerja.

Catatan: IE/edge 15 dan versi yang lebih lama tidak mendukung sticky

Hola Aku adalah sticky
